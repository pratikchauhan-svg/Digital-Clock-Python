# Digital-Clock-Python
This project is a Digital Clock application developed using Python and Tkinter. The clock displays the current system time in hours, minutes, and seconds and updates automatically every second. The project demonstrates the use of Tkinter for GUI development and Python's time module for real-time functionality. 

## Features
- Real-time clock display
- Updates every second
- GUI built with Tkinter

## Technologies Used
- Python
- Tkinter
- Time Module
